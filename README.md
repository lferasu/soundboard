# Super Soundboard (React Native)

This project is now a **React Native app** built with **Expo** (not a static website).

## Features

- 8 large, kid-friendly sound buttons.
- Voice-based sound effects using `expo-speech`.
- Works on iOS, Android, and Expo web.

## Project structure

- `App.js` - Main React Native soundboard UI and interaction logic.
- `app.json` - Expo app configuration.
- `babel.config.js` - Babel config for Expo.
- `package.json` - Dependencies and scripts.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start Expo:

```bash
npm run start
```

3. Open in:
- Expo Go on a phone, or
- Android/iOS simulator.

## Notes

- This app uses text-to-speech (`expo-speech`) to generate the sound effects.
- No static website files are required anymore.
